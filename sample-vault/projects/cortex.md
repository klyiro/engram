# Cortex

An agent-first, markdown-native second brain: a dashboard for humans and an MCP server for
agents, over a git-backed folder of notes.

## Principles
- Markdown files are the source of truth. No database.
- Git is the durable store.
- Agents ([[ada-lovelace|and humans]]) read and write the same notes.
